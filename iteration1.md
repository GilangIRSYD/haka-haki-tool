Design a modern financial analytics web dashboard for Indonesian stock market users.

The website focuses on **broker accumulation analysis for a specific stock (emiten)**.

### Input Section (Top Area):

* Multi-select input for **Broker Codes** (example: AK, BK, YU)
* Text input for **Stock Code / Emiten** (example: FORE)
* Date range picker (From – To, example: 2025-11-01 to 2025-12-30)
* Primary CTA button: **"Analyze"**

### After user clicks Analyze:

#### 1. Executive Summary Section

* Summary card explaining whether selected brokers are in **Accumulation or Distribution phase**
* Include:
  * Net Buy / Net Sell total
  * Short insight text (example: “Brokers AK & YU show strong accumulation during the selected period”)
* Use green color for net buy, red for net sell

#### 2. Calendar View (Main Visualization)

* Large monthly calendar layout
* Latest month shown at the top, previous months stacked below
* Each date cell shows:
  * Net lot value
  * Net transaction value (IDR)
* Green background for net buy days
* Red background for net sell days
* Neutral gray for no transaction

### UI Style:

* Clean, modern fintech dashboard
* Dark mode friendly
* Professional Indonesian stock trading feel
* Clear typography and strong data hierarchy

### UX Notes:

* Smooth transitions when loading data
* Calendar is scrollable
* Responsive for desktop and tablet

Output high-fidelity UI design.
